# Свадебное приглашение

Красивая интерактивная открытка-приглашение на свадьбу.

## Как использовать

1. **Добавьте фотографию молодожен:**
   - Поместите вашу фотографию в папку проекта
   - Назовите файл `couple-photo.jpg` (или измените путь в `index.html`)
   - Рекомендуемый размер: 400x300px или больше (сохранит пропорции)

2. **Настройте информацию:**
   - Откройте `index.html`
   - Измените имена молодожен (строка 26)
   - Измените дату свадьбы (строка 27)
   - Измените время, место и контактную информацию (строки 38-48, 54)

3. **Откройте в браузере:**
   - Просто откройте файл `index.html` в любом браузере
   - Нажмите кнопку "Открыть приглашение" для просмотра содержимого

## Структура файлов

- `index.html` - основная структура страницы
- `style.css` - стили оформления
- `script.js` - интерактивность (переворот карточки)
- `couple-photo.jpg` - фотография молодожен (добавьте свою)

## Особенности

- ✨ Интерактивная карточка с эффектом переворота
- 📸 Место для фотографии молодожен
- 💌 Красивое сопроводительное письмо
- 📱 Адаптивный дизайн для всех устройств
- 🎨 Элегантный дизайн в пастельных тонах
